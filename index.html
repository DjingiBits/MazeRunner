Welcome to Maze Runner!

  1. To start the game: Press “Play Game”
  2. To move press the Up/Down/Left/Right arrow keys.
  3. You exit the maze, when all gems are collected.
  4. One correct answer equals one gem.
  5. If the provided answer is not correct, you lose 30points.
  6. If the provided answer is not correct, you can "buy" the gem for 300points
  7. For every correct answer you get +100points.
  8. If the points become 0 game is over.
